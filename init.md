### Create a new repository on the command line
```
touch README.md
git init
git add README.md
git commit -m "First commit"
git remote add origin https://github.com/pkadima/NAME.git
git push -u origin master
```

### Push an existing repository from the command line
```
git remote add origin https://github.com/pkadima/NAME.git
git push -u origin master
```
